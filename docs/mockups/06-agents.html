Step 2 · Endpoints

Three endpoints, designed for agents.

GET
/api/v1/reports

List original compliance reports — metadata only. Use this to discover which slug to pay for. Rate limit: 60 req/min/IP.

Free
no payment
GET
/api/v1/reports/:slug

Full markdown content of one report. Settled via x402 on Base mainnet in USDC at the price returned by the list endpoint (default $0.10, per-report priceUsd overrides).

$0.10
x402 · base
POST
/api/v1/freeze/diagnose

Structured diagnosis for a bank-card freeze scenario. Returns risk_level + 7-step action plan + document checklist + lawyer questions. The disclaimer field is mandatory — surface to end user verbatim.
