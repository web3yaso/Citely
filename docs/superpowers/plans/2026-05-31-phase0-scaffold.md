# Phase 0 — Scaffold Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up a Next.js 16 App Router project at the repo root that boots with `pnpm dev`, renders a home shell, and has a working **Connect MetaMask** button (wagmi v2 + viem `injected`, no RainbowKit / WalletConnect).

**Architecture:** Single Next.js project at repo root (PRD §6 "单 project"). Wallet connect layer is isolated in `lib/wagmi-config.ts` + `components/shared/WalletConnect.tsx` so EAS/x402 signing code (later phases) sits on viem and the connector can be swapped later. Design tokens come from `docs/mockups/01-home.html`'s inline palette, wired as Tailwind v4 `@theme` tokens.

**Tech Stack:** Next.js 16 (App Router, Turbopack), React 19, TypeScript 5, Tailwind v4, pnpm, wagmi v2 + viem + `@tanstack/react-query`, Vitest.

**Series context:** This is plan 1 of the build series (Phase 0). Subsequent plans: Phase 1 home port, Phase 2 EAS, Phase 3 MDX, Phase 4 x402, Phase 4.5 companion, Phase 5 leaderboard, Phase 6 deploy. See `docs/HACKATHON.md` §11 and `docs/superpowers/specs/2026-05-31-Citely-build-design.md` §3.

```ts
/** Shorten an EVM address to `0x1234…5678`. Throws if not a 0x-prefixed 40-hex address. */
export function truncateAddress(address: string): string {
  if (!/^0x[0-9a-fA-F]{40}$/.test(address)) {
    throw new Error(`Invalid EVM address: ${address}`);
  }
  return `${address.slice(0, 6)}…${address.slice(-4)}`;
}
```

- [ ] **Step 1: Create `components/shared/WalletConnect.tsx`**

`components/shared/WalletConnect.tsx`:
```tsx
"use client";

import { useAccount, useConnect, useDisconnect } from "wagmi";
import { injected } from "wagmi/connectors";
import { truncateAddress } from "@/lib/format";

export function WalletConnect() {
  const { address, isConnected } = useAccount();
  const { connect, isPending } = useConnect();
  const { disconnect } = useDisconnect();

  if (isConnected && address) {
    return (
      <button
        onClick={() => disconnect()}
        className="rounded-md bg-charcoal px-4 py-2 font-mono text-sm text-paper-card hover:bg-charcoal-2"
      >
        {truncateAddress(address)} · Disconnect
      </button>
    );
  }

  return (
    <button
      onClick={() => connect({ connector: injected({ target: "metaMask" }) })}
      disabled={isPending}
      className="rounded-md bg-crimson px-4 py-2 text-sm font-semibold text-white hover:bg-crimson-hi disabled:opacity-60"
    >
      {isPending ? "Connecting…" : "Connect MetaMask"}
    </button>
  );
}
```
