# Phase 4 — x402 paywall (real unlock) + agent endpoint

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`).

**Goal:** Make the paywall real. `GET /api/v1/articles/[slug]` returns 402 → pay USDC on Base Sepolia → 200 with full markdown + companion 〔A〕. The Human + Agent paywall CTAs unlock the full text via the connected wallet; AI agents (AgentCash) hit the same endpoint.

**Architecture:** One x402-protected route serves both humans and agents. `lib/x402-server.ts` builds the `x402ResourceServer` (CDP facilitator + `ExactEvmScheme`); the route uses `withX402` with **dynamic `payTo`/`price` read per-slug from the attestation index** (on-chain-verified at publish). Slug whitelist runs BEFORE the paywall (invalid → 404). The handler (runs only after payment verifies) decrypts the body (`getReportBody`) + paid companion 〔A〕 (`getCompanionPaidZone`), appends to `data/payment-log.json`, and returns them. Browser side: `lib/x402-client.ts` wraps `fetch` with payment from the wagmi wallet (`@x402/fetch` + `@x402/evm`); a client `UnlockGate` swaps the 24% preview for the full content on success and caches it in localStorage (per §5 Story 3 "永久可读" via client cache).

**Confirmed v2 API (from installed type defs):**
- Server: `withX402(handler, routeConfig, server)`; `routeConfig.accepts` is a `PaymentOption { scheme, network, payTo: string | (ctx)=>string|Promise, price: Price | (ctx)=>Price|Promise }`; `ctx: HTTPRequestContext { path, method, ... }`. `server = new x402ResourceServer(facilitatorClient).register("eip155:84532", new ExactEvmScheme())`. Facilitator: `new HTTPFacilitatorClient(createFacilitatorConfig(CDP_API_KEY_ID, CDP_API_KEY_SECRET))`.
- Client: `registerExactEvmScheme(new x402Client(), { signer })` then `wrapFetchWithPayment(fetch, client)`. `signer: ClientEvmSigner` = a viem WalletClient extended with `publicActions` (bridge from wagmi `useWalletClient()`).
- Price accepts a `"$0.30"` USD string; the scheme maps it to USDC on the network.

**Deferrals:** the leaderboard wiring to `payment-log.json` is Phase 5 (this phase just WRITES the log). "永久可读" is client-cache only (per §5 Story 3 hackathon simplification; server stays stateless).

**Env (all present):** `CDP_API_KEY_ID`, `CDP_API_KEY_SECRET`, `X402_FACILITATOR_URL`, `X402_NETWORK=eip155:84532`, `NEXT_PUBLIC_USDC_ADDRESS`, `NEXT_PUBLIC_EAS_SCHEMA_UID`, `CONTENT_ENC_KEY`. Packages `@x402/next @x402/fetch @x402/evm @x402/core @x402/paywall @coinbase/x402` (2.14.0) installed.

**Series:** Plan 6 (Phase 4). Spec: HACKATHON §5 Story 3 & 4, §8. Prior phases provide `getReportBody`, `getCompanionPaidZone`, `attestation-index`, the paywall UIs.

```ts
import { readFileSync, writeFileSync, renameSync, existsSync } from "node:fs";
import { resolve } from "node:path";

const LOG_PATH = resolve(process.cwd(), "data/payment-log.json");
export type PaymentEntry = { slug: string; payer: string; amount: string; txHash: string; ts: number };

export function readPaymentLog(): PaymentEntry[] {
  if (!existsSync(LOG_PATH)) return [];
  return JSON.parse(readFileSync(LOG_PATH, "utf8"));
}
export function appendPaymentLog(e: PaymentEntry): void {
  const all = readPaymentLog();
  all.push(e);
  const tmp = `${LOG_PATH}.tmp`;
  writeFileSync(tmp, JSON.stringify(all, null, 2) + "\n");
  renameSync(tmp, LOG_PATH);
}
```

```ts
import { x402ResourceServer } from "@x402/next";
import { HTTPFacilitatorClient } from "@x402/core/server";
import { ExactEvmScheme } from "@x402/evm/exact/server";
import { createFacilitatorConfig } from "@coinbase/x402";
import { findRecord } from "./attestation-index";

export const X402_NETWORK = "eip155:84532";

// Network safety: never run mainnet config with testnet keys (§8.5).
if (process.env.X402_NETWORK && process.env.X402_NETWORK !== X402_NETWORK) {
  throw new Error(`X402_NETWORK must be ${X402_NETWORK}, got ${process.env.X402_NETWORK}`);
}

let _server: x402ResourceServer | null = null;
export function getX402Server(): x402ResourceServer {
  if (!_server) {
    const facilitator = new HTTPFacilitatorClient(
      createFacilitatorConfig(process.env.CDP_API_KEY_ID, process.env.CDP_API_KEY_SECRET),
    );
    _server = new x402ResourceServer(facilitator).register(X402_NETWORK, new ExactEvmScheme());
  }
  return _server;
}

export function slugFromPath(path: string): string {
  return path.replace(/\/+$/, "").split("/").pop() ?? "";
}

/** payTo = on-chain author (recorded at publish, EAS-verified). Throws if unknown. */
export function payToForSlug(slug: string): string {
  const rec = findRecord(slug);
  if (!rec) throw new Error(`no published record for ${slug}`);
  return rec.author;
}

/** price as a USD string ("$0.30") derived from the on-chain priceUSDC. */
export function priceUsdForSlug(slug: string): string {
  const rec = findRecord(slug);
  if (!rec) throw new Error(`no published record for ${slug}`);
  return "$" + (Number(BigInt(rec.priceUSDC)) / 1e6).toFixed(2);
}
```

### Task 2: `app/api/v1/articles/[slug]/route.ts` (withX402)

**Files:** `app/api/v1/articles/[slug]/route.ts`

- [ ] **Step 1: Implement** — slug whitelist BEFORE paywall; dynamic payTo/price; handler returns full content + companion 〔A〕 + citation and logs the payment:
```ts
import { NextRequest, NextResponse } from "next/server";
import { withX402 } from "@x402/next";
import { getX402Server, X402_NETWORK, slugFromPath, payToForSlug, priceUsdForSlug } from "@/lib/x402-server";
import { getReportBody, getReportMeta } from "@/lib/reports";
import { getCompanionPaidZone } from "@/lib/companions";
import { findRecord } from "@/lib/attestation-index";
import { appendPaymentLog } from "@/lib/payment-log";

const SLUG_RE = /^[a-z0-9-]{1,80}$/;

const handler = async (req: NextRequest): Promise<NextResponse> => {
  const slug = slugFromPath(new URL(req.url).pathname);
  const meta = getReportMeta(slug);
  const rec = findRecord(slug)!;
  const content = getReportBody(slug);          // full markdown (decrypted, server-only)
  const companion = getCompanionPaidZone(slug); // paid 〔A〕 zone (decrypted)
  // The handler only runs after the facilitator verified payment → record it for the leaderboard.
  appendPaymentLog({ slug, payer: req.headers.get("x-payer") ?? rec.author, amount: rec.priceUSDC, txHash: rec.attestationUID, ts: Date.now() });
  return NextResponse.json({
    slug,
    title: meta.title,
    content,
    companion,
    citation: { author: meta.authorName, attestationUID: rec.attestationUID, publishedAt: meta.publishedAt },
  });
};

const paid = withX402(
  handler,
  {
    accepts: {
      scheme: "exact",
      network: X402_NETWORK,
      payTo: (ctx) => payToForSlug(slugFromPath(ctx.path)),
      price: (ctx) => priceUsdForSlug(slugFromPath(ctx.path)),
    },
    mimeType: "application/json",
    description: "Citely — paid article full text + companion",
  },
  getX402Server(),
);

export async function GET(req: NextRequest, { params }: { params: Promise<{ slug: string }> }) {
  const { slug } = await params;
  // §8: slug whitelist BEFORE the paywall. Invalid/unpublished → 404, never 402.
  if (!SLUG_RE.test(slug) || !findRecord(slug)) {
    return NextResponse.json({ error: "not found" }, { status: 404 });
  }
  return paid(req);
}

export const dynamic = "force-dynamic";
```

(b) §8.3: the 402 body must NOT contain the markdown — `withX402` returns its own 402 (payment requirements only); our handler only runs on 200, so the full text never appears in a 402. Do NOT add an `unpaidResponseBody` that leaks content.

- [ ] **Step 1: Implement:**
```ts
import { x402Client } from "@x402/core/client";
import { registerExactEvmScheme } from "@x402/evm/exact/client/register";
import { wrapFetchWithPayment } from "@x402/fetch";
import { publicActions, type WalletClient } from "viem";

export type ArticlePaid = {
  slug: string;
  title: string;
  content: string;
  companion: string;
  citation: { author: string; attestationUID: string; publishedAt: string };
};

/** Build a fetch that auto-pays x402 using the connected wagmi wallet. */
export function makePaidFetch(walletClient: WalletClient) {
  const signer = walletClient.extend(publicActions);
  const client = new x402Client();
  registerExactEvmScheme(client, { signer: signer as never });
  return wrapFetchWithPayment(globalThis.fetch, client);
}

/** Pay for + fetch the full article. */
export async function unlockArticle(walletClient: WalletClient, slug: string): Promise<ArticlePaid> {
  const paidFetch = makePaidFetch(walletClient);
  const res = await paidFetch(`/api/v1/articles/${slug}`);
  if (!res.ok) throw new Error(`unlock failed: ${res.status}`);
  return (await res.json()) as ArticlePaid;
}
```

```tsx
"use client";
import { useState, useEffect } from "react";
import { useAccount, useConnect, useWalletClient } from "wagmi";
import { injected } from "wagmi/connectors";
import { unlockArticle, type ArticlePaid } from "@/lib/x402-client";

const cacheKey = (slug: string) => `citely_unlocked_${slug}`;

export function UnlockGate({
  slug, priceUsd, authorName, preview, renderFull,
}: {
  slug: string; priceUsd: string; authorName: string;
  preview: React.ReactNode;                     // server-rendered 24% preview + paywall card
  renderFull: (full: ArticlePaid) => React.ReactNode; // client renderer for the full article
}) {
  const { isConnected } = useAccount();
  const { connect } = useConnect();
  const { data: walletClient } = useWalletClient();
  const [full, setFull] = useState<ArticlePaid | null>(null);
  const [status, setStatus] = useState<"idle" | "paying" | "error">("idle");
  const [err, setErr] = useState<string | null>(null);

  useEffect(() => {
    const cached = localStorage.getItem(cacheKey(slug));
    if (cached) { try { setFull(JSON.parse(cached)); } catch {} }
  }, [slug]);

  async function onUnlock() {
    setErr(null);
    if (!isConnected) { connect({ connector: injected({ target: "metaMask" }) }); return; }
    if (!walletClient) return;
    setStatus("paying");
    try {
      const data = await unlockArticle(walletClient, slug);
      localStorage.setItem(cacheKey(slug), JSON.stringify(data));
      setFull(data);
      setStatus("idle");
    } catch (e) { setErr((e as Error).message); setStatus("error"); }
  }

  if (full) return <>{renderFull(full)}</>;
  return (
    <div>
      {preview}
      {/* the CTA is provided by preview's paywall card; intercept it via a button this gate renders instead */}
      <div className="unlock-actions">
        <button className="pw-cta" onClick={onUnlock} disabled={status === "paying"}>
          {status === "paying" ? "付款中…" : !isConnected ? `连接钱包付 ${priceUsd}` : `用钱包付 ${priceUsd} 解锁全文`}
        </button>
        {err && <p className="pw-fine" style={{ color: "var(--crimson)" }}>{err}</p>}
      </div>
    </div>
  );
}
```
